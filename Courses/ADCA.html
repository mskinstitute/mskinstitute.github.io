Course Description

ADCA, Advance Diploma in Computer Application, is a diploma course (18 MONTHS ) in computer applications that include the study of numerous computer applications such as MS Office, Internet applications, operating systems, accounting, graphics such as Photoshop, Coral Draw, and web content such as HTML and CSS among other topics.
